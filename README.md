# [Car Sales Analysis Dashboard]

## 🎯 Project Overview
This project focuses on analyzing Car sales to identify key trends, performance bottlenecks, and growth opportunities. 
The project were part of the Udemy course named Data Analysis | SQL, Tableau, Power BI, Excel | Real Projects. 
This dashboard was created using a dataset downloaded from the course, but also available in Kaggle.

* **Live Dashboard:** https://public.tableau.com/app/profile/fabi.n.vargas.ruiz/viz/CarSalesAnalysis_17824413624370/Dashboard1?publish=yes

---

## 🛠️ Tech Stack & Tools
* **Data Visualization:** Tableau Public
* **Data Processing:** Excel, however in this particular project, data was used as it were on the xlsx
* **Dataset Source:** Kaggle / Udemy Course. I downloaded from the course files available.

---

## 🚀 Key Insights & Business Value
* **Insight 1:** Identified that even when electric vehicles are getting popular, still the traditional vehicles are the most popular by far.
* **Insight 2:** Despise that fact, we were able to confirm Tesla is the brand that sells the most, being all of the electric cars.
* **Insight 3:** King is the county in which most of the sales has been done.

---

## 📈 Dashboard Features & Architecture
The final Tableau workbook consists of a primary dashboard:
1. EV Analysis focused on Washington State, in the United States. Different charts where can be seen sales by vehicle type,
which brands were the most popular, in which counties were these vehicles sold; you can check by numbers and in a map for visual
understanding and as well a comparition through time. A particular car brand may be filtered to review data related to that.


---

## 🔧 Data Preparation Process
In this case, the dataset provided were not cleaned or transformed in any way.
Formulated custom metrics using Tableau LOD (Level of Detail) expressions and window calculations to track rolling averages.
